Estrutura Básica

- tag Doctype -> definição do documento.
- tag HTML -> Documento HTML.
- tag Head -> Cabeça -> onde serão definidas as informações do documento.(O usuário nao enxerga)
- tag Title -> Título do site que aparece na aba do navegador - tag Body -> Corpo -> Onde serão definidos o conteúdo do documento(O que o usuário ira enxergar)
